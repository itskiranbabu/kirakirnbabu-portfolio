## üõ† Essential MCP Servers

### Databases & Storage
*   **[Airtable](https://github.com/mcp-server/airtable)**: Full read/write access to Airtable bases and records.
*   **[PostgreSQL](https://github.com/mcp-server/postgres)**: Direct SQL execution and schema inspection for Postgres clusters.
*   **[Cloudflare KV](https://github.com/cloudflare/mcp-server-cloudflare)**: Low-latency Key-Value storage orchestration.

### Web & Browser Automation
*   **[Puppeteer MCP](https://github.com/mcp-server/puppeteer)**: High-level API to control headless Chrome for scraping and testing.
*   **[Playwright](https://github.com/mcp-server/playwright)**: Multi-browser automation for complex RAG pipelines.
*   **[Brave Search](https://github.com/mcp-server/brave-search)**: Real-time web search capabilities for private AI browsing.

### Developer Tools
*   **[GitHub MCP](https://github.com/mcp-server/github)**: Inspect issues, PRs, and repository metadata natively.
*   **[Docker MCP](https://github.com/mcp-server/docker)**: Control local container lifecycles from within agentic workflows.
*   **[GitLab](https://github.com/mcp-server/gitlab)**: Enterprise-grade CI/CD and repository management.

### Productivity & Communication
*   **[Slack](https://github.com/mcp-server/slack)**: Send messages, search history, and manage channels.
*   **[Notion](https://github.com/mcp-server/notion)**: Manipulate workspaces, pages, and databases via AI.
*   **[Google Drive](https://github.com/mcp-server/google-drive)**: Dynamic file retrieval and document processing.

## üè¢ Enterprise Patterns
1.  **Security First**: Use RBAC (Role-Based Access Control) at the proxy layer before exposing MCP servers to autonomous agents.
2.  **Stateless Execution**: Design servers to be stateless to scale horizontally in production environments.
3.  **Observability**: Integrate with OpenTelemetry to track every tool call made by the LLM.
